## 🚀 使用流程

### 完整用户旅程

```
1. 访问系统 (http://localhost:3001)
   ↓
2. 注册/登录
   ↓
3. 创建项目
   - 输入项目名称
   - 描述目标（自然语言）
   ↓
4. 系统自动执行
   - 解析目标 (5秒)
   - 生成团队 (2秒)
   - 规划任务 (10秒)
   - 执行任务 (30-120秒/任务)
   ↓
5. 实时监控
   - 查看Agent状态
   - 跟踪任务进度
   - 阅读沟通记录
   - 查看决策过程
   ↓
6. 获取结果
   - 生成的代码
   - 技术文档
   - 决策记录
   - 完整追溯
```

**输入**:
```
项目名称: Task Management API
目标: Build a REST API for task management with FastAPI and PostgreSQL. 
Include CRUD endpoints for tasks, user authentication, and basic documentation.
```
